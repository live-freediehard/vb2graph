VB2Gram Usage -
=================

 
Use in command prompt :

>VB2Gram.exe InputFileName#InputFilePath


It will generate graph file



VB2Graph Usage -
==================

Use in command prompt :

>VB2Graph.exe InputGraphFile#InputGraphFile#GraphvizbinPATH#OutputDirPath


Sample : 
Inp_FileName - "test.frmabc"
